## Shortcuts for Productivity

**Create New React Application**:

```bash
# npm 7+, extra double-dash is needed:
npm create vite@latest <name> -- --template react
cd <name>
npm install
npm run dev
```

**User code Snippets**
* Config file is at `/.vscode/user.code-snippets`
```
log: console.log('');
rc: React Component
rce: React Class Component  
```
